Working with Arrays


Arrays

An array will store multiple values in a single variable.

var cars = ['Volkwagen', 'Audi', 'Mercedes', 'BMW'];
console.log(cars.length);

The fun of working with arrays is the values can be added or removed through various methods in order to restructure the array.
Arrays are zero-indexed, meaning the first position is identified as 0, and the last position is the length of the full array minus 1. In the case below, the array's length is (4-1=3).

Arrays will always preserve ordering, which means they will remember the order in which the items were added.



Starting with an empty array

An array is assigned similarly to a variable, but assigning the brackets (even if nothing is within them) is needed to identify it as an array.

var arraySet = [];
console.log(arraySet);

Keep in mind that setting an existing array to an empty set will clear any items within it. It can be used as a clear reset option.



Indexing into an Array

To access an individual item in an array, we call into it by referencing the array variable name, proceeded by [i], where i is the index of the position whose value we're seeking.

var zoo = ['zebra', 'rhino', 'penguin', 'monkey'];
console.log(zoo[2]);

Add item(s) to the end

Using push, items can be added to the array to the end of the array.

var zoo = ['zebra', 'rhino', 'penguin', 'monkey'];
zoo.push('panda');
console.log(zoo);

Remove item from the end

Using pop, the last item of the array can be removed and moved into a new variable.

var zoo = ['zebra', 'rhino', 'penguin', 'monkey', 'panda'];
var freeAnimal = zoo.pop();
console.log(zoo);

Add item(s) to the beginning

Using unshift will place the data at the beginning of the array.

var zoo = ['zebra', 'rhino', 'penguin', 'monkey'];
zoo.unshift('emu');
console.log(zoo);

Remove item from the beginning

Using shift will remove the item from the beginning of the array.

var zoo = ['emu', 'zebra', 'rhino', 'penguin', 'monkey'];
var freeAnimal = zoo.shift();
console.log(zoo);

Overwrite item at specific index

When you need to replace data over an existing value, you can simply assign the existing position with new content.

var meals = ['breakfast', 'lunch', 'dinner'];
meals[1] = 'brunch';
console.log(meals);

Copy an Array

Using slice is a great method for copying all of the data of an array into a new one.

var nums = [23, 18, 9, 2];
var numsCopy = nums.slice();
console.log(numsCopy);

Join()

If you were asked to add all of the items of an array into a single string, there are a few different ways you can accomplish this. This first way simply takes each item from the array and adds it into the string. Although this works, compare this technique with the join() method (see below) which is much easier to write.

var campingGear = ["Tent", "Axe", "Sleeping Pad", "Toliet Paper"];
var bringOnTrip = campingGear[0] + ' and ' + campingGear[1] + ' and ' + campingGear[2] + ' and ' + campingGear[3];
console.log(bringOnTrip);

The join() method joins all of the elements of an array and places them into a string, and then returns the string.

var campingGear = ["Tent", "Axe", "Sleeping Pad", "Toliet Paper"];
var bringOnTrip = campingGear.join();
console.log (bringOnTrip);

As you can see from the returned string in the above example, each item is separated by a comma by default. A unique separator can be added to this method by passing the custom separator within the join method.

var campingGear = ["Tent", "Axe", "Sleeping Pad", "Toliet Paper"];
var bringOnTrip = campingGear.join(" and ");
console.log (bringOnTrip);

Modifying an Array with splice()

Using splice will insert values within an existing array.

nums.splice(3, 1, "Item1", 44, "Item5");
// 3 is the position we'll place new items.
// 1 is how many items we're removing.
// "Item1", 44, "Item5" are all items being added into the array.



var pizzaToppings = ["Pepperoni", "Mushrooms", "Cheese"];
pizzaToppings.splice(3, 0, "Tomato Sauce", "Dough");
// 1 is the position we'll place new items.
// 0 is how many items we're removing.
// 99, 33 are the items we're adding.
console.log('Pizza Toppings: ' + pizzaToppings);

Removing Items from an Array

The splice() method can also be used to remove items from an array.
Keep in mind how the first two numbers are used within a splice:

nums.splice(3, 1);
// 3 is the position we'll place new items.
// 1 is how many items we're removing.

By not adding new items into the array, the position and the removal numbers work perfectly to delete items for an Array.

var animals = ["Pig", "Goat", "Duck", "Cow", "Chicken", "Sheep"]; animals.splice(3, 1);
console.log ('Farm Animals: ' + animals)
var pets = ['Dog', 'Cat', 'Turtle', 'Bird'];
pets.splice(0, 3);
console.log('Pets: ' + pets);

Placing Arrays within Arrays #1

Placing arrays within arrays can be a simple as treating arrays as if they are variables.

var birds1 = ['hawk', 'eagle', 'falcon'];
var birds2 = ['duck', 'duck', 'goose'];
var birdsTotal = [ birds1, birds2];
console.log('Birds: ' + birdsTotal);

Placing Arrays within Arrays #2

Splice can be used to insert arrays within existing arrays.

var birds = ['hawk', 'eagle', 'falcon'];
birds.splice(2, 0, ['duck', 'duck', 'goose']);
console.log('Birds: ' + birds);

Rearranging with sort()

The sort function can seem easy for how powerful it can be in rearranging arrays in both alphabetical or numerical orders. It can also order the data in ascending (up) or descending (down).


Sorting Numbers

To sort numbers in ascending order:

var points = [45, 102, 1, 3, 27, 12];
points.sort(function(a, b) {
return a - b
});
console.log (points);

To sort numbers in decending order:

var points = [45, 102, 1, 3, 27, 12];
points.sort(function(a, b) {
return b - a
});
console.log (points);

Certain numbers can easily be pulled from an array. Simple arrange the order first, and then select the appropriate item. This example below will select the hight number in the array.

var points = [45, 102, 1, 3, 27, 12];
points.sort(function(a, b) {
return b - a
});
console.log (points[0]);

Sorting Strings

When an array is sorted alphabetically, it will always be sorted A - Z then a-z.

var books = ["To Kill a Mockingbird ", "Memoirs of a Geisha", "Lord of the Flies", "Charlotte's Web "];
books.sort();
console.log (books);

If you wish to reverse this order, you'll need to use the reverse function,

var books = ["To Kill a Mockingbird ", "Memoirs of a Geisha", "Lord of the Flies", "Charlotte's Web "];
books.sort();
books.reverse();
console.log (books);

Determining the Length of Strings

This relatively simple method will return the number of characters in a string.

var importantString = "Hello World!";
var stringLength = importantString.length;
console.log (stringLength);
